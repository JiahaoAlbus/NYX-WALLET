## Description (Long)
NYX WALLIET is a non-custodial, multi-chain wallet built for iOS 17+. It supports Ethereum, BNB Smart Chain, Bitcoin, Solana, and TRON with secure on-device key management using Secure Enclave and Keychain. 

Features include biometric signing, seed sharding, transaction risk checks, swap quotes, NFT portfolio visibility, and a clean multi-language interface. You control your keys and funds at all times.
